# FPGA Lab Experiments (Verilog)

This repository contains all FPGA lab experiments implemented using Verilog HDL with simulation results.

## Tools Used

* Icarus Verilog
* GTKWave
* Vivado (for FPGA implementation)

## Experiments Covered

* Full Adder
* 4-bit Adder
* Encoder & Decoder
* Multiplexer & Demultiplexer
* Counters
* Shift Registers
* FSM (Mealy & Moore)
* ALU
* CPU

## How to Run

```bash
iverilog file.v testbench.v
vvp a.out
gtkwave dump.vcd
```
